# Stakeholder Call Transcript

## Topic: Subscriptions, Credits, and ADS on Keylead Platform

**Goal:** Subscriptions = baseline exposure, Ads = premium discovery, Credits = flexible upgrades. Separate subscriptions from credits. Introduce Ads in April.

### 1. The Keylead Value Ladder

Subscription tiers (starter, growth, pro, plus). Agents join Keylead to:
- Get discovered
- Upload listings
- Generate content
- Schedule promotions
- Get exposure
- Get leads
- Close deals

**Promotions** — What the subscription powers. Listings get distributed through:
- Keylead discovery
- Instagram channel promotions

**Credits** — Power-ups to upgrade/make changes:
- Upgrade promotion to bigger account
- Move promotion earlier in calendar
- Extra post outside plan
- Additional graphic revisions
- Feature in newsletter
- Upgrade listing exposure

**Discovery Ads** — Agents can dominate discovery ranking (e.g., "Featured Miami Agent"). Generates inbound leads.

### 2. How Subscriptions Influence Ranking

Weighted ranking formula:
```
DiscoveryScore = Relevance + Content Quality + Engagement + SubscriptionScore + BoostScore
```

- SubscriptionScore = small advantage
- BoostScore = large advantage

Effect:
- Free user → normal ranking
- Starter → slight priority
- Growth → stronger priority
- Pro → higher priority
- Plus → strongest organic ranking
- Ads dominate on top

### 3. The Simplified Keylead Economy (3 Layers)

**Layer 1 = Subscriptions** — Unlock platform exposure capacity:
- How many promotions per month (pre-built templates)
- Access to channels
- Automation tools (content creation, AI-schedule)
- Analytics (overview)

**Layer 2 = Discovery Ads** — Premium discovery ranking:
- Two products: Partner Boost and Property Boost
- Premium inventory, not subscription perks
- Only 3 partner slots per city

**Layer 3 = Credits** — Flexible upgrades for extra actions:
- Upgrade to bigger channel
- Move promotion earlier
- Request additional revisions
- Buy extra promotions
- Upgrade exposure

### 4. Removing Credits from Subscriptions

Including credits in subscriptions causes confusion, unused credits, complicated pricing.
Removing them will: simplify onboarding, make subscriptions clearer, increase credit purchases.
Credits become intent-based purchases.

### 5. Discovery Ads are Separate

Only 3 partner slots per city. 100 PRO users in Miami but only 3 Discovery Boost slots. Subscriptions cannot guarantee ad placement.

### 6. Why Subscriptions Influence Ranking

Even without ads, paying users should rank slightly higher organically. Creates subscription value beyond Instagram promo benefits.

### 7. The Value Flywheel

Subscriptions → Agents → Listings → Content → Traffic → Discovery demand → Ad revenue

Only paying subscribers have discovery ads access (increases subscription conversions).

### Action Items for Product Dev

Visually separate three concepts inside the product with three tabs:
1. **Exposure** (subscriptions)
2. **Boost** (discovery ads)
3. **Upgrades** (credits)

### What Credits Should NOT Control

Credits should NOT control core platform usage:
- Uploading listings
- Generating content
- Basic scheduling
- Basic exposure

Those belong to subscriptions.

### User Spending Behaviors

- Light users → subscriptions only
- Active marketers → subscriptions + credits
- Market leaders → subscriptions + credits + discovery boost
