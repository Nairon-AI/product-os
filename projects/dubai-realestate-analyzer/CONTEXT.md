# Dubai Real Estate Analyzer Context

## Overview
Personal investment intelligence tool that tracks Dubai real estate prices over time — across buildings, neighborhoods, and off-plan projects — to enable data-backed purchase decisions during a market downturn. The core product is the dataset itself, queried primarily through Claude Code.

## Target Users
Personal investor (non-professional) with cash ready to deploy during a crisis-driven price decline. Looking at off-plan projects under 500k AED with a 2-year investment horizon.

## Product Shape
- **Platform:** Data-first — structured dataset with Claude Code as primary query interface
- **Interaction Model:** Dataset + CLI/Claude Code analytics, optional lightweight web dashboard
- **Key entities:** Projects (buildings), Neighborhoods, Listings, Price snapshots over time

## Boundaries
- NOT a listing portal — no buying/agent flow
- NOT a SaaS product — personal use only
- NOT a full analytics platform — focused on one investment decision
- Data collection method TBD (scraping, API, manual entry)
